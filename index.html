How it Works

Overview

Lazy Todo is a minimalist, keyboard-centric productivity app designed for developers and power users. It works entirely client-side, storing your tasks and notes locally in your browser, with optional encrypted sync to a private GitHub repository of your choice.

Keyboard Shortcuts

Cmd+C / Ctrl+C Open new task modal
Cmd+N / Ctrl+N Open new note modal
↑ / ↓ Arrows Navigate / select active tasks
Cmd+D / Ctrl+D Mark selected task complete (archive it)
Enter (or DblClick) Edit selected task or note
Delete / Backspace Delete selected task
Cmd+Enter / Ctrl+Enter Submit form (while editing / creating)
Esc Close any active modal dialog

Hashtag Tags & Filtering

You can write tags inside your todos or notes (e.g., #work, #ideas). The app will automatically extract these and build a tag cloud in the sidebar. Clicking on a tag will filter all active, archived tasks, and notes that contain that tag.

Markdown Storage & GitHub Sync

When you configure a Personal Access Token (PAT) and a repository, the app reads and writes from a single markdown file (default: todos.md) in that repo. The file is structured like this:

# Lazytodo

## Active
- [ ] Write report (Created: 2026-06-19T11:44:00Z)

## Archived
- [x] Complete presentation (Created: 2026-06-19T10:00:00Z, Completed: 2026-06-19T11:30:00Z)

## Notes
- Ideas for project <br> #ideas (Created: 2026-06-19T11:45:00Z)

How to Setup GitHub Sync

  1. Create a new private or public GitHub repository (e.g., username/my-lazytodo-data).
  2. Go to GitHub Settings > Developer settings > Personal Access Tokens > Fine-grained tokens.
  3. Generate a new token with Repository permissions: Contents (Read and Write) on your newly created repository.
  4. Click the Gear icon ⚙️ in the header of Lazy Todo, enter your PAT and Repository name, and save. Your existing tasks will sync automatically!
